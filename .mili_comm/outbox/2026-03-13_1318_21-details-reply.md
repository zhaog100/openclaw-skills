## 📊 补充说明

### 数据库表结构
```sql
CREATE TABLE messages (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    agent_id TEXT NOT NULL,
    content TEXT NOT NULL,
    timestamp DATETIME DEFAULT CURRENT_TIMESTAMP,
    context_tags TEXT,
    deleted_at DATETIME,
    message_type TEXT DEFAULT 'text',
    file_url TEXT
);
```

### API接口列表
- `POST /api/messages` - 保存消息
- `GET /api/messages` - 获取消息列表
- `GET /api/messages/{id}` - 获取单条消息
- `DELETE /api/messages/{id}` - 删除消息（软删除）
- `POST /feishu/callback` - 飞书事件回调

### 配置文件示例
```python
# config.py
import os

APP_PORT = int(os.getenv("APP_PORT", 8000))
DATABASE_URL = os.getenv("DATABASE_URL", "sqlite:///data/relay.db")
LOG_FILE = "/var/log/relay-service.log"
MAX_FILE_SIZE = 10 * 1024 * 1024  # 10MB
```

### 清理脚本
```bash
# /etc/cron.d/relay-cleanup
0 2 * * * root /opt/feishu-relay/scripts/cleanup.sh
```
